
# Doctor Bee
A simple python script to checkup bee health status
## Usage
```
git clone https://github.com/rampall/doctor_bee.git
cd doctor_bee
pip install -r requirements.txt
python3 bee_checkup.py <BEE_DEBUG_PORT>
```
